Форматы документов

.DOC или .doc ❲от англ. document❳ - расширение имени файла, используемое для файлов, представляющих текст, с разметкой или без. Расширение .Doc часто использовалось для обозначения простых текстовых файлов без оформления, однако позже стало использоваться для двоичных форматов с резметкой.
Документ MS Word
Portable Document Format ❲ PDF ❳ - межплатформенный формат электронных документов, разработанный фирмой Adobe Systems с использыванием ряда возможностей языка Postscript.
Документ PDF
ZIP - популярный формат сжатия данных и архивации файлов. Файл в этом формате обычно имеет раширение .zip и хранит в сжатом или несжатом виде один или несколько файлов, которое можно из него извлечь путём распаковки с помощью специальной программы.
